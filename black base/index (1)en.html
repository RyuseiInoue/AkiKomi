Abstract


　"Aki" means vacant, and "Komi" means crowded.

　Corona virus that’s been going rampant has changed our life completely. We are forced to stay away from each other and put on a mask everywhere we go.
　So, we came up with the system that could provide you with the information of congestion status of a room in real time. We named our system “Akikomi”.

　We materialize the idea by using the sensor counting system and link the information to the smartphone application so that users can choose where to go without concerning about corona virus.
　It reduces the risk of corona virus because it decreases the chance to make a room congested and prevent people to gather in one place.









Objective

　Today's biggest goal is to complete this system quickly and start using it in school.
　Aiming to suppress the cluster of new coronavirus, we will build a system that can grasp the degree of congestion of people in the room in real time.

　In the future, we aim to have the data obtained from this system utilized in sites where clusters are likely to occur.
　We will proceed with the project with a view to utilizing it in local restaurants and public facilities.


Purpose of each team
Software engineer team We will develop an application that allows users to check room availability information using Xcode.
Designer team We will create a website for companies.
Electronic engineer team We will make a sensor to count the comings and goings of people.
